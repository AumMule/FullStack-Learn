## ⚡ How to Compile and Run the Exercises

To run any of the exercises from a terminal:
1.  Navigate into the specific exercise folder.
2.  Compile all Java files.
3.  Run the corresponding test file.

### Commands:
```bash
# Navigate to the exercise directory
cd "Module 1 - Design Patterns and Principles/SingletonPatternExample"

# Compile all classes
javac *.java

# Run the test client
java SingletonTest
```

## 📦 Detailed Exercise Breakdown

### 💎 Exercise 1: Singleton Pattern
*   **Scenario:** Ensure that a logging utility class (`Logger`) has only one instance throughout the application lifecycle to guarantee consistent logging.
*   **Key Concept:** Private constructor to prevent instantiation, and double-checked locking inside a public static method for thread safety.

#### 📁 Files:
*   [Logger.java](file:///c:/Users/ommul/OneDrive/Desktop/cognizant/Module%201%20-%20Design%20Patterns%20and%20Principles/SingletonPatternExample/Logger.java)
*   [SingletonTest.java](file:///c:/Users/ommul/OneDrive/Desktop/cognizant/Module%201%20-%20Design%20Patterns%20and%20Principles/SingletonPatternExample/SingletonTest.java)

#### 💻 Core Code (Logger):
```java
public class Logger {
    private static Logger instance;
    private Logger() {} // Private Constructor

    public static Logger getInstance() {
        if (instance == null) {
            synchronized (Logger.class) {
                if (instance == null) {
                    instance = new Logger();
                }
            }
        }
        return instance;
    }
}
```
